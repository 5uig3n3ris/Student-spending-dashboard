Student Spending Dashboard

Documentation

Project Name: Student Spending Dashboard

Features:

  • Interactive Doughnut, Bar, and Line Charts with Chart.js
  • Real-time AI insights based on spending patterns
  • CSV and PDF Export by academic year and filter range
  • Custom form to add spending categories and amounts
  • Filter data by academic year and custom date range

Use Case: Helps students visualize and analyze budgeting habits with a user-friendly dashboard and insights.
